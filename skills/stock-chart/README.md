# 股票折线图技能

## 📈 功能简介
这是一个用于查询A股股票实时价格并显示折线图的OpenClaw技能。用户可以通过输入股票名称或代码，获取实时数据并生成今天开盘到现在的价格折线图。

## 🚀 快速开始

### 安装依赖
```bash
pip install requests matplotlib numpy
```

### 基本使用
```bash
# 进入技能目录
cd skills/stock-chart

# 查询贵州茅台
python stock_chart_cn.py 贵州茅台

# 使用股票代码
python stock_chart_cn.py 600519

# 查询上证指数
python stock_chart_cn.py 上证指数
```

## 📊 功能特点

### 核心功能
- ✅ **实时数据查询**: 从新浪财经获取A股实时价格
- ✅ **折线图显示**: 显示今天开盘到现在的价格走势
- ✅ **纯中文界面**: 所有提示信息均为中文
- ✅ **自动保存**: 图表自动保存为PNG图片
- ✅ **涨跌可视化**: 红色表示上涨，绿色表示下跌

### 支持的股票类型
- **A股个股**: 贵州茅台、五粮液、宁德时代、比亚迪等
- **银行股**: 招商银行、工商银行、建设银行等
- **保险股**: 中国平安
- **指数**: 上证指数、深证成指、创业板指
- **所有A股**: 支持任意6位A股代码

## 🎨 图表展示

### 图表元素
1. **标题区域**: 股票名称 + 当前价格 + 涨跌幅
2. **折线图**: 今天交易时间段的价格走势
3. **关键价格线**: 
   - 橙色虚线: 今日开盘价
   - 蓝色虚线: 昨日收盘价
   - 红色虚线: 今日最高价
   - 绿色虚线: 今日最低价
4. **信息框**: 成交量、成交额、更新时间
5. **涨跌区域**: 红色/绿色填充表示涨跌区域

## 🔧 技术实现

### 数据源
- **API**: 新浪财经免费接口 `https://hq.sinajs.cn/list=<股票代码>`
- **数据格式**: CSV格式，包含30+个字段
- **更新频率**: 实时更新（约3秒一次）

### 图表技术
- **库**: matplotlib + numpy
- **模拟数据**: 基于实时数据生成合理的分时走势
- **字体支持**: 自动检测系统字体，支持中文显示

   ```bash

   # 验证股票代码格式
   # 确认股票在交易时间内
   ```

   ```bash

   # 或使用非交互模式保存图片
   ```
