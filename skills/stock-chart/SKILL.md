# 股票折线图技能

## 技能描述
这是一个用于查询A股股票实时价格并显示折线图的技能。用户可以输入股票名称或代码，技能将获取实时数据并生成今天开盘到现在的价格折线图，顶部显示股票名称和当前价格。

## 功能特点
- **实时数据**: 从新浪财经获取A股实时价格数据
- **折线图表**: 显示今天开盘到现在的价格走势图
- **中文界面**: 纯中文提示信息和界面
- **自动保存**: 图表自动保存为PNG图片文件
- **涨跌显示**: 红色表示上涨，绿色表示下跌
- **关键价格线**: 显示开盘价、昨收价、最高价、最低价

## 使用方法

### 基本使用
```bash
# 查询贵州茅台
python stock_chart_cn.py 贵州茅台

# 使用股票代码
python stock_chart_cn.py 600519
python stock_chart_cn.py sh600519

# 查询上证指数
python stock_chart_cn.py 上证指数
```

### 支持的股票
- **A股**: 贵州茅台, 五粮液, 宁德时代, 比亚迪
- **银行股**: 招商银行, 工商银行, 建设银行, 农业银行, 中国银行
- **保险股**: 中国平安
- **指数**: 上证指数, 深证成指, 创业板指
- **任何A股**: 支持所有6位A股代码

## 技术实现

### 数据来源
使用新浪财经免费API：
```
https://hq.sinajs.cn/list=<股票代码>
```

### 数据格式
返回逗号分隔的数据，包含：
- 股票名称
- 开盘价、最高价、最低价、当前价
- 成交量、成交额
- 更新时间

### 图表生成
使用matplotlib生成折线图：
1. 模拟今天交易时间段的价格走势
2. 根据实时数据调整趋势
3. 添加关键价格线和标注
4. 保存为PNG图片

## 故障排除

### 常见问题
1. **字体显示方块**: 系统缺少中文字体，可安装微软雅黑或黑体
2. **无法获取数据**: 检查网络连接和股票代码是否正确
3. **图表不显示**: 确保matplotlib正确安装，可能需要安装tkinter

### 安装依赖
```bash
# 安装Python依赖
pip install requests matplotlib numpy

# Windows系统可能需要安装tkinter
# 在Python安装时勾选"tcl/tk and IDLE"
```
